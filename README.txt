1-Installation instructions: Install the plugin by following these steps:

Log in to your WordPress admin dashboard.
Go to "Plugins" -> "Add New" -> "Upload Plugin."
Choose the zip file provided to you and click "Install Now."
Once installed, click "Activate Plugin" to activate it.

2-Shortcode usage:
Follow the instruction below:

Inside a post or page editor, you can use the shortcode by enclosing it in square brackets. For example: [pubg_name_generator].
